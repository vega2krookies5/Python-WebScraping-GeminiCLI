# Yes24 AI 도서 데이터 분석 및 시각화

## 1. 분석 목표
- `yes24_ai.csv` 데이터를 사용하여 AI 분야 베스트셀러 도서의 특징을 파악합니다.
- 가격, 출판사, 발행일 등의 분포를 확인하고, 주요 변수 간의 관계를 시각화하여 인사이트를 도출합니다.

## 2. 데이터 불러오기 및 전처리
- `pandas` 라이브러리를 사용하여 CSV 파일을 불러옵니다.
- 데이터의 결측치를 확인하고, 분석에 적합한 형태로 데이터 타입을 변환합니다.
- '정가', '판매가', '리뷰 수', '판매지수' 컬럼을 숫자형으로 변환합니다.
- '발행일'에서 연도와 월을 추출하여 새로운 컬럼을 생성합니다.


## 3. 탐색적 데이터 분석(EDA) 및 시각화
- yes24 폴더 안에 시각화 이미지를 images 폴더를 생성해서 저장할 것 
- **기초 통계 분석:** 범주형, 숫자형 데이터의 기술통계를 확인합니다.
- **수치 데이터 분석:** 모든 수치 데이터의 분포를 히스토그램으로 시각화합니다.
- **범주형 데이터 분석:** 모든 범주형 데이터의 빈도수를 구하고 막대그래프로 시각화 합니다. 범주가 너무 많을 때는 상위 20개만 시각화 합니다. 
- **출판사 분석:** 가장 많은 책을 출판한 상위 출판사 20개를 가로 막대 그래프로 시각화합니다.
- **발행 트렌드 분석:** 연도별, 월별 도서 발행 빈도수 트렌드를 시각화합니다.
- **상관 관계 분석:** 가격, 리뷰 수, 판매지수 간의 상관 관계를 히트맵으로 시각화합니다.
- **워드 클라우드:** 도서 제목을 형태소 분석하여 워드 클라우드로 시각화합니다. 한글폰트가 깨지지 않게 설정합니다. 이때, 너무 오래 걸리는 형태소 분석은 하지 말아주세요. 
- 2~3개의 변수를 사용한 교차표와 피봇테이블 분석을 5개 이상 해주세요.

## 분석결과 마크다운 파일로 저장
- 모든 분석 결과를 마크다운 파일로 저장하고, 분석 결과에 대한 인사이트를 함께 작성할 것 
- info, 수치형, 범주형 기술 통계 내용과 시각화 내용에 대한 교차표나 피봇 테이블도 함께 구하고 모두 마크다운 파일에 저장할 것
